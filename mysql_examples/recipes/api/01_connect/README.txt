This directory contains recipes showing how to connect to the MySQL
server. The languages demonstrated are:

Java:
  Connect.java

PHP:
  connect.php

Perl:
  connect.pl

Python:
  connect.py

Ruby:
  connect.rb

The "connect2" scripts in this directory show alternative methods for
connecting to the server, generally by specifying a TCP/IP port number or
Unix domain socket file value other than the default.
